Dockernized whitebaord
======================

Prepare
-------
1. [install docker-compose](https://docs.docker.com/compose/install/)
2. build container image

```
docker-compose build
```

Run
---
1. run

```
docker-compose up -d
```
2. access http://localhost:3000

Stop
----
1.

```
docker-compose down
```


Configuration
--------------
* Listen port number
  * modify ports part of docker-compose.yml
